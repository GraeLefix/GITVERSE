# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a Vulnerability

**Please do not open a public issue for security vulnerabilities.**

Report via [GitHub Security Advisories](https://github.com/GraeLefix/GITVERSE/security/advisories).

We respond within 48 hours and will work with you to resolve issues responsibly before public disclosure.

## Security Scope

- API key exposure in logs or responses
- Rate limiting bypasses
- Prompt injection via malicious repository content
- Data exfiltration via crafted input
